# Ollama Chatbot

A simple terminal-based chatbot using Python and Ollama with llama3.

## Requirements

- Python 3.x
- Ollama running locally
- llama3 model

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python chatbot.py
```

Type your messages and get responses from llama3. Type 'quit', 'exit', or 'bye' to exit.
